## 👨‍💻 Sobre mim

Tenho 22 anos e estudo **Análise e Desenvolvimento de Sistemas na PUC Goiás**.

Sou apaixonado por desenvolvimento de software, automação e criação de soluções que resolvam problemas reais.

Atualmente, concentro meus estudos e projetos nas seguintes áreas:

- ☕ Desenvolvimento backend com Java e Spring Boot.
- 📱 Aplicativos mobile com Flutter e Dart.
- 💻 Sistemas desktop para Windows.
- 🐧 Automação com Linux, Bash e ferramentas de terminal.
- 🌐 Criação e integração de APIs REST.
- 🗄️ Bancos de dados relacionais e serviços em nuvem.
- 📊 Sistemas de gestão voltados para negócios.
- 🤖 Ferramentas que utilizam automação e inteligência artificial.

Meu objetivo é evoluir constantemente como desenvolvedor e construir aplicações modernas, confiáveis, intuitivas e úteis.
